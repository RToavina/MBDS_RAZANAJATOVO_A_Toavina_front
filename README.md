### Installation 

1- Clôner le repo Github 

```sh
git clone https://github.com/RToavina/MBDS_RAZANAJATOVO_A_Toavina_front.git
```

2- Installer les dépendances

```sh
npm install
```

3- Lancer l'app
```
ng serve
```

4- Accéder au [lien](http://localhost:4200/)  : `http://localhost:4200/`
